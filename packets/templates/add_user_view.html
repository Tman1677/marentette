Adding users

To add a user you're going to need to go to the admin page and login with the SeaholmPhysicsStudent account I gave Mr. Stukey. That account is a super user and can do anything to the database storing the video links. Although you can create another super user account I wouldn't get in the habit of it or give super user accounts out willy nilly to students. Because of that this guide assumes you're going to make a student account that can only add videos, not sabotage other work.

On the admin page click plus next to users, give them a username and then let them type in a password. Click save to go to the next page. The next page looks really complicated but there's only two things we need to do.
